Sync & Strike is a turn-based fighting game inspired by Your Only Move Is Hustle. In this game, players fights 2-on-1 with an AI companion against another AI in a 2D arena. The system uses frame data similar to traditional fighting games. At the beginning of a match, The player choose an action, and afterward the game determines whose turn it is based on the chosen move and the number of remaining frames.

In a turn, you can choose one of the following:
- movement options
- attack options
- special options
- super options
- defense options
